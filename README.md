# FedoraBugCrawler

```
python buggrabber.py -h
usage: buggrabber.py [-h] -d D [-v V]
                     [-s {NEW,ASSIGNED,ON_DEV,POST,MODIFIED,ON_QA,VERIFIED,RELEASE_PENDING,CLOSED}]

Use the script to pull Fedora bugzilla issues.

optional arguments:
  -h, --help            show this help message and exit
  -d D, -desc D         Short description of the bug
  -v V, -version V      Fedora product version
  -s {NEW,ASSIGNED,ON_DEV,POST,MODIFIED,ON_QA,VERIFIED,RELEASE_PENDING,CLOSED}, -bugstatus {NEW,ASSIGNED,ON_DEV,POST,MODIFIED,ON_QA,VERIFIED,RELEASE_PENDING,CLOSED}
                        Status of the bug
```
